## Description

The-flower-power-project est un projet développé en parallèle de la formation OpenClassRooms :
"Développeur intégrateur web".

## Installation

```bash
cd the-flower-power-project
npm install
npm run dev
```

- Jest
- Redux, Redux-Toolkit
- Git, GitHub
